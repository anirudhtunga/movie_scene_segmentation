# Video Scene Segmentation Using Local and Global Attention

## Introduction

The task is to predict scene transition boundaries of a movie when features are provided for each shot. This is an implementation of multi-modal transformer using local and global attentions to predict the scene boundaries. 

## Model Training


* Arrange the data in the data folder.
* Python version >= 3.6
* Local Attention Window size can be changed in model.py
* To train the model: ``` python train.py```
